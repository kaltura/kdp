getMediaTypeTimelineTypes()method 
public static function getMediaTypeTimelineTypes(media_type:uint):uint

et the given media type possible timeline types. ie. MediaTypes.IMAGE can only be added to TimelineTypes.VIDEO, and MediaTypes.VIDEO can be added to either TimelineTypes.VIDEO or TimelineTypes.AUDIO.

Parameters
media_type:uint — the media type to check.

Returns
uint — the possible hosting timeline types.

translateServerType()method 
public static function translateServerType(int_type:uint, to_server:Boolean = false):uint

ranslate server media type to MediaTypes type, our values are organized to be used as bit mask.

Parameters
int_type:uint — the media type given by the server on getAllEntries method.

Returns
uint — the local MediaType.
translateStringTypeToInt()method 
public static function translateStringTypeToInt(str_type:String):uint

ranslates string type to uint type to be used in the sdk. p>SDL uses string types to be maintain readability, so we need to convert it when parsing the sdl to roughcut timelines.

Parameters

Returns
