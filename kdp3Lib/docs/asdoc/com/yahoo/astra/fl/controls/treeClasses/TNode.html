Packagecom.yahoo.astra.fl.controls.treeClasses
Classpublic dynamic class TNode
SubclassesBranchNode, LeafNode

The TNode class is the base class for three different types of node objects that may appear in a Tree component data provider. The TNode class provides the necessary logic for linking the nodes hierarchically, recording the information about the nesting level of a node, and checking whether a node is currently visible.
