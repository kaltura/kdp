Property detail
artistproperty
artist:String  [read-only]

Gets the artist for the audio clip (read-only). This information is available:

  • If the SWF is in the same domain as the sound file.
  • If the checkPolicyFile property is true and there is a cross-domain policy file on the server from which the sound is loaded that permits the domain of the loading SWF file.
